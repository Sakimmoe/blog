Pulse探针的安装与升级

2025-12-29 技术

前两天在论坛看到这个探针，非常符合我的审美，我非常喜欢这种简洁干净的风格。

Pulse的安装

一键安装

curl -fsSL https://raw.githubusercontent.com/xhhcn/Pulse/main/install-pulse-server.sh | sudo bash

脚本会自动：
✅ 检测系统架构

✅ 下载对应的二进制文件

✅ 配置 systemd 服务

✅ 启动服务并设置开机自启

访问 http://YOUR_IP:8008 查看监控面板

Pulse的升级

先删掉原来的版本

systemctl stop pulse-server && sudo systemctl disable pulse-server 
rm -f /opt/pulse/pulse-server /etc/systemd/system/pulse-server.service
systemctl daemon-reload

再次运行一次安装脚本

curl -fsSL https://raw.githubusercontent.com/xhhcn/Pulse/main/install-pulse-server.sh | sudo bash

升级完成！

Pulse的删除

客户端删除命令

systemctl stop pulse-client && systemctl disable pulse-client && rm -rf /opt/pulse/pulse-client /etc/systemd/system/pulse-client.service

服务端删除命令

systemctl stop pulse-server && systemctl disable pulse-server && rm -rf /opt/pulse /etc/systemd/system/pulse-server.service
